This file translates attributes depending upon the source of the infrasturcture.

For example, the original lab on AWS there are:
* openshift_kubeadmin_password

But on Infra CNV, there are:
* openshift_cluster_admin_password
* openshift_cluster_admin_username

So, we translate
 :openshift_kubeadmin_password: {openshift_cluster_admin_password}

Note that kubeadmin is removed from the Infra CNV.
So we need to write more complex logic to translate the attribute.

////

:openshift_kubeadmin_password: {openshift_cluster_admin_password}
endif::openshift_cluster_admin_username[]
ifndef::openshift_cluster_admin_username[]
:openshift_kubeadmin_username: kubeadmin
:openshift_kubeadmin_password: {openshift_kubeadmin_password}
endif::openshift_cluster_admin_username[]
